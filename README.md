# Godoy Auto Detailing Website

Static website project for Godoy Auto Detailing.

## Open Locally

During this Codex session, the site preview is running at:

```text
http://localhost:4181
```

You can also open `index.html` directly in a browser because this is a static website.

## Important Files

- `index.html` - main website file with HTML, CSS, and JavaScript.
- `assets/` - logo, hero images, and gallery images.
- `scripts/` - helper scripts used to generate/process gallery and cover images.
- `PROJECT_PIPELINE.md` - timeline of project changes and implementation notes.
